## 📝 Sobre o Projeto
O **My Habits** é uma aplicação mobile desenvolvida em React Native para auxiliar no acompanhamento de rotinas diárias. O projeto foca em uma interface limpa (Clean UI) e uma experiência de usuário fluida, permitindo o gerenciamento de hábitos essenciais de forma prática.

---

## 🚀 Funcionalidades
- **Check-in de Hábitos:** Interface interativa para marcar tarefas concluídas no dia.
- **Modais Dinâmicos:** Fluxo amigável para adição de novos hábitos.
- **Feedback Visual:** Uso de estados para indicar o progresso das atividades.
- **Design Responsivo:** Adaptado para diferentes tamanhos de tela mobile.
